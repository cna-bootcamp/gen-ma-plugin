DMAP 스킬 활성화 경로를 보여주는 기술 다이어그램.

상단: "사용자 요청" 시작점

두 갈래 분기:

왼쪽 경로 "직접 활성화":
- 사용자 → "슬래시 명령 또는 명확한 자연어" → 런타임이 직접 매칭 → 특정 스킬 활성화
- 예시: "/abra:scenario" → scenario 스킬 바로 실행
- 파란색 실선 화살표

오른쪽 경로 "Core 경유 활성화":
- 사용자 → "모호한 요청" → 런타임 → Core 스킬 활성화 → 의도 판별 → 적절한 스킬로 라우팅
- 예시: "에이전트 만들어줘" → Core → 전체 워크플로우 스킬
- 주황색 점선 화살표

하단: 5가지 스킬 유형이 나란히 배치
- Core (보라색) — "행동 규범 + 라우팅"
- Setup (초록색) — "설치/설정 (직결형)"
- Planning (파란색) — "계획 수립"
- Orchestrator (주황색) — "워크플로우 지휘"
- Utility (회색) — "보조 기능 (직결형)"

위임형(Core, Planning, Orchestrator)은 실선 테두리
직결형(Setup, Utility)은 점선 테두리

스타일: 흐름도 스타일, 깔끔한 플랫 디자인, 파스텔 색상, 한글 텍스트. 흰색 배경.
